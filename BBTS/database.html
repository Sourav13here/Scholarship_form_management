Database Structure

Nucleon Scholarship Application System

Database Information

Database Type: SQLite

Database File: scholarship.db

Location: Same directory as the PHP files

Purpose: Stores all scholarship application data submitted through the form

Table: applications

This table stores all scholarship application submissions

Column Name Data Type Constraints Description
id INTEGER PRIMARY KEY, AUTO INCREMENT Unique identifier for each application
application_id TEXT UNIQUE, NOT NULL Human-readable application ID (e.g., NCI20240001)
name TEXT NOT NULL Applicant's full name
class TEXT NOT NULL Current class of the student
school TEXT NOT NULL Name of the school
address TEXT NOT NULL Full address of the applicant
contact TEXT NOT NULL Primary contact number (10 digits)
alt_contact TEXT OPTIONAL Alternative contact number
email TEXT NOT NULL Email address of the applicant
achievements TEXT OPTIONAL Previous achievements in Science and Math
declaration INTEGER NOT NULL Declaration acceptance (1 = agreed)
submission_date DATETIME DEFAULT CURRENT_TIMESTAMP Date and time of submission
status TEXT DEFAULT 'pending' Application status

How to Access the Database

1. Using Admin Panel: Visit admin-panel.php to view all applications with statistics

2. Using SQLite Browser: Download and install DB Browser for SQLite, then open scholarship.db

3. Using PHP: The database is automatically managed through the PHP backend files

4. Command Line: Use sqlite3 scholarship.db to access via terminal
