METHOD 1: USING COMMAND LINE (RECOMMENDED)
================================================================================

STEP 1: Open Command Prompt
---------------------------
1. Press Windows + R
2. Type: cmd
3. Press Enter

STEP 2: Navigate to Your Project
--------------------------------
cd "s:\growtez\1.2 Nucleon Scholarship"

STEP 3: Initialize Git Repository
---------------------------------
git init

STEP 4: Add All Files
--------------------
git add .

STEP 5: Create First Commit
---------------------------
git commit -m "Initial commit: Nucleon Scholarship Application System"

STEP 6: Create GitHub Repository
--------------------------------
1. Go to: https://github.com/
2. Click the "+" icon (top-right)
3. Click "New repository"
4. Repository name: nucleon-scholarship-system
5. Description: Scholarship Application Management System for Nucleon Coaching Institute
6. Choose: Public or Private
7. DO NOT initialize with README (we already have files)
8. Click "Create repository"

STEP 7: Link Local Repository to GitHub
---------------------------------------
Replace YOUR_USERNAME with your GitHub username:

git remote add origin https://github.com/YOUR_USERNAME/nucleon-scholarship-system.git

STEP 8: Push to GitHub
----------------------
git branch -M main
git push -u origin main

STEP 9: Enter GitHub Credentials
--------------------------------
- Username: Your GitHub username
- Password: Use Personal Access Token (not your password)

To create Personal Access Token:
1. GitHub ‚Üí Settings ‚Üí Developer settings ‚Üí Personal access tokens ‚Üí Tokens (classic)
2. Generate new token
3. Select scopes: repo (all)
4. Copy the token
5. Use it as password when pushing

================================================================================
TROUBLESHOOTING
================================================================================

Error: "fatal: not a git repository"
Solution: Run: git init

Error: "remote origin already exists"
Solution: Run: git remote remove origin
Then add again: git remote add origin URL

Error: "Permission denied"
Solution: Use Personal Access Token instead of password

Error: "Large files"
Solution: Check .gitignore, remove large files

Error: "Merge conflicts"
Solution: Pull first: git pull
Then resolve conflicts and push
