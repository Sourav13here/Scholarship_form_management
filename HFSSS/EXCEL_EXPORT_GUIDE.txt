================================================================================
TROUBLESHOOTING
================================================================================

Problem: Excel file not downloading
Solution:
  - Check if logged in as admin
  - Clear browser cache
  - Try different browser
  - Check PHP error logs

Problem: Formatting not showing
Solution:
  - Open in Microsoft Excel (not Notepad)
  - Try Excel Online or Google Sheets
  - Re-download the file

Problem: Date shows ### in Excel
Solution:
  - This is fixed in the new format!
  - Date now shows as: 2024-10-24 16:30
  - Fits in standard column width

Problem: Special characters broken
Solution:
  - File uses UTF-8 encoding
  - Should work in all modern Excel versions
  - Try opening in Google Sheets

Problem: Both buttons not showing
Solution:
  - Refresh the admin panel page
  - Clear browser cache
  - Check admin-panel.php is updated
